# Quick Start
1. Clone the repository:

   ```shell
   git clone https://github.com/VyLeet/laypy.git
   ```
3. Move to the script's directory:

   ``` shell
   cd laypy
   ```
4. Add `execute` mode:
   
   ```shell
   chmod +x laypy.sh
   ```
5. Run the script with Python packages as arguments:

   ```shell
   ./laypy.sh pandas altair
   ```
